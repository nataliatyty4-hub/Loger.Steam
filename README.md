loger — это утилита для автоматической диагностики проблем с модами и играми в Steam.

Утилита:

Находит установленный Steam

Определяет игру по APP_ID

Проверяет наличие логов и ошибок модов

Автоматически сообщает, что работает, а если что-то не так — выводит готовые решения

Полностью автономная: exe не требует исходного loger.py или дополнительных файлов

Системные требования

Windows 10 / 11

Python 3.12 (для сборки exe)

Установленный Steam

Для запуска exe Python не нужен.

Установка

Скачать exe-файл loger.exe

Поместить его в любую папку

Дважды кликнуть для запуска

Важно: exe полностью автономный, исходный loger.py не нужен.

Использование

Запустите loger.exe

Введите APP_ID игры (если используется консольная версия)

Дождитесь результатов диагностики:

✅ Успешно → всё в порядке

❌ Не получилось → показаны возможные решения

Примеры проблем, которые проверяет утилита:

Steam не найден

Игра не найдена по APP_ID

Логи недоступны

Ошибки с модами

Советы

Если exe запускается в OneDrive, лучше переместить его в обычную папку (например, C:\loger_build\)

Для GUI-версии консоль не используется, и ввод происходит через окна

Сборка из исходников (для разработчиков)

Если нужно собрать exe самостоятельно:

Убедитесь, что установлен Python 3.12

Установите PyInstaller:

python -m pip install pyinstaller


Перейдите в папку с loger.py и выполните:

python -m PyInstaller --onefile loger.py


exe появится в папке dist\loger.exe

Теперь loger.py можно удалить — exe полностью автономный.
